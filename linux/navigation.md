# Linux Navigation & File Management

## Commands

### Show current directory
pwd

### List files
ls
ls -l   # detailed view
ls -a   # show hidden files

### Change directory
cd /path
cd ..   # go back

### Create directories
mkdir folder_name

### Remove directories
rmdir folder_name

### Create files
touch file.txt

### Remove files/directories
rm file.txt
rm -rf folder_name

---

## Key Concepts

- Linux is case-sensitive
- Most work is done via terminal
- Navigation is essential for all DevOps tasks

## Practice Example

mkdir test
